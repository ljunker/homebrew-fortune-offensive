# Ljunker Fortune-offensive

## How do I install these formulae?

`brew install ljunker/fortune-offensive/<formula>`

Or `brew tap ljunker/fortune-offensive` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "ljunker/fortune-offensive"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
